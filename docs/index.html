Agora

Agora is a project that allows anyone to sell files on the web for bitcoin using the Lightning Network. Everything on this site is served by an Agora instance run by the Agora developers. Agora is free and open source, so anyone can run their own Agora instance.

Agora's code is hosted on GitHub.

Feedback is greatly appreciated. If you run into a bug or have a feature request, feel free to open an issue.

We publish periodic updates to our blog.
